# Optimización SEO con Next.js: Guía Completa 2024

El SEO (Search Engine Optimization) es fundamental para el éxito de cualquier sitio web. Next.js ofrece herramientas poderosas para optimizar tu sitio y mejorar su posicionamiento en buscadores.

## ¿Por qué Next.js es Excelente para SEO?

Next.js está diseñado pensando en el SEO desde el principio:

- **Server-Side Rendering (SSR)** - Los motores de búsqueda pueden leer tu contenido
- **Static Site Generation (SSG)** - Sitios súper rápidos
- **Meta tags dinámicos** - Control total sobre la metadata
- **Sitemap automático** - Fácil indexación
- **Optimización de imágenes** - Mejor rendimiento

## Configuración Básica de SEO

### 1. Metadata en Layout
```jsx
// app/layout.tsx
export const metadata = {
  title: {
    default: "Mi Sitio Web",
    template: "%s | Mi Sitio Web"
  },
  description: "Descripción de mi sitio web",
  keywords: ["palabra1", "palabra2"],
  openGraph: {
    title: "Mi Sitio Web",
    description: "Descripción de mi sitio web",
    url: "https://misitio.com",
    siteName: "Mi Sitio Web",
    images: [
      {
        url: "/og-image.jpg",
        width: 1200,
        height: 630,
        alt: "Mi Sitio Web",
      },
    ],
    locale: "es_ES",
    type: "website",
  },
  twitter: {
    card: "summary_large_image",
    title: "Mi Sitio Web",
    description: "Descripción de mi sitio web",
    images: ["/og-image.jpg"],
  },
}
```

### 2. Metadata por Página
```jsx
// app/about/page.tsx
export const metadata = {
  title: "Acerca de Nosotros",
  description: "Conoce más sobre nuestra empresa y equipo",
  openGraph: {
    title: "Acerca de Nosotros | Mi Sitio Web",
    description: "Conoce más sobre nuestra empresa y equipo",
    url: "https://misitio.com/about",
  },
}
```

## Estructura de URLs SEO-Friendly

### Rutas Dinámicas
```jsx
// app/blog/[slug]/page.tsx
export async function generateStaticParams() {
  const posts = getAllPosts();
  return posts.map((post) => ({
    slug: post.slug,
  }));
}
```

### Redirecciones
```jsx
// next.config.js
async redirects() {
  return [
    {
      source: '/old-page',
      destination: '/new-page',
      permanent: true,
    },
  ];
}
```

## Sitemap y Robots.txt

### Sitemap Automático
```jsx
// app/sitemap.ts
export default function sitemap() {
  return [
    {
      url: 'https://misitio.com',
      lastModified: new Date(),
      changeFrequency: 'yearly',
      priority: 1,
    },
    {
      url: 'https://misitio.com/blog',
      lastModified: new Date(),
      changeFrequency: 'weekly',
      priority: 0.8,
    },
  ];
}
```

### Robots.txt
```txt
User-agent: *
Allow: /

Sitemap: https://misitio.com/sitemap.xml
```

## Schema Markup (Datos Estructurados)

```jsx
const structuredData = {
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Mi Empresa",
  "url": "https://misitio.com",
  "logo": "https://misitio.com/logo.png",
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+1234567890",
    "contactType": "customer service"
  }
};

// En tu componente
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{ __html: JSON.stringify(structuredData) }}
/>
```
